### GitHub Flavored Markdown

GitHub Flavored Markdown (GFM) is the dialect of Markdown that is currently supported for user
content on GitHub.com and GitHub Enterprise. The [GFM Spec](gfm-spec.md) page contains the full
specification, imported from the [official GFM spec site](https://github.github.com/gfm/){:target="_blank"}.

!!! note "Regenerating the spec page"

    The `gfm-spec.md` file is generated by running the fetch script from the repository root:

    ```bash
    pip install -r scripts/fetch-gfm-spec-requirements.txt
    python scripts/fetch-gfm-spec.py
    ```
